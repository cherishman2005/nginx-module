# nodejs

* 回调太多难于控制（俗称回调地狱）和 CPU 密集任务处理的不是很好。
  -- 目前异步流程技术已经取得了非常不错的进步，从Callback、Promise 到 Async函数，可以轻松的满足所有开发需求。
  
  * CPU 密集任务处理并非不可解，方案有很多:
    * 比如通过系统底层语言 Rust 来扩展 Node.js。
    * nodejs嵌入C++ module插件。

## 安装yarn

```
npm install -g yarn
```
